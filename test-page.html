Troubleshooting:

  • If the extension doesn't load, check the console for errors in chrome://extensions/
  • Make sure all files are present in the extension directory
  • Verify that the manifest.json is valid JSON
  • Check if Chrome's experimental AI features are enabled (if available)
  • Reload the extension after making any changes
